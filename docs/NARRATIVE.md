# 剧情与角色设计

## 1. 故事梗概

玩家扮演一名现代大学生林疏影，在一次雷雨夜被闪电击中，意识模糊间转生到修仙世界，成为天剑宗刚入门的小师妹。

天剑宗位于云海深处的浮空山脉，表面上是正道魁首，镇压魔道、庇护苍生。然而随着玩家逐渐深入宗门生活，发现全宗上下皆是"反派"：
- 大师兄萧寒修炼禁术《噬灵剑诀》，以生灵精血铸剑
- 二师姐红绡掌管丹堂，以修士炼制人丹
- 师父云深子竟是千年前被封印的魔道巨擘"云深魔君"

系统任务是"成为正道之光"，但身边全是反派。玩家需要在"维持正派 façade"与"揭露真相"之间做出选择，最终走出自己的"剑道"。

## 2. 核心主题

- **剑无正邪，唯心所向**：呼应标题"不知剑"，强调选择的意义
- **伪善与真恶**：正道与魔道的界限是否如此分明？
- **系统与自由意志**：被强加的"正道任务"vs 自己的选择

## 3. 主要角色

### 3.1 玩家角色：林疏影

- **身份**：转生者，天剑宗小师妹
- **性格**：由玩家选择塑造（初始为好奇、谨慎）
- **特殊能力**：作为转生者，能看到他人头顶的"因果线"（暗示其真实身份）
- **秘密**：前世与云深魔君有莫大渊源

### 3.2 师父：云深子 / 云深魔君

- **表面**：仙风道骨的正道宗师，深受弟子爱戴
- **真实**：千年前被正道联手封印的魔道巨擘，借尸还魂建立天剑宗
- **对玩家的态度**：似乎知道玩家的转生身份，有意培养
- **秘密**：建立天剑宗是为了积蓄力量，向当年封印他的正道复仇

### 3.3 大师兄：萧寒

- **身份**：云深子首徒，天剑宗大师兄
- **表面**：冷峻正直，宗门楷模
- **真实**：修炼禁术《噬灵剑诀》，每杀一人便强一分
- **动机**：幼时全族被魔道所灭，发誓要以任何手段变强报仇
- **与玩家关系**：初期冷漠，若玩家选择理解他则逐渐敞开心扉

### 3.4 二师姐：红绡

- **身份**：丹堂执掌者
- **表面**：温柔美艳，擅长炼丹救人
- **真实**：以修士炼制人丹，甚至将失败品"赐"给外门弟子
- **动机**：天生绝脉无法修炼，只能靠人丹续命
- **与玩家关系**：表面热情，实则试探，若玩家威胁到她则毫不留情

### 3.5 三师兄：墨言

- **身份**：阵法天才，掌管宗门禁地
- **性格**：书卷气重，不善言辞，似乎知道很多但不说
- **立场**：中立派，既不认同也不揭发，只守护自己在乎的人
- **与玩家关系**：最容易成为盟友的角色，但需要证明玩家的"不同"

### 3.6 四师姐：白芷

- **身份**：最小的师姐，比玩家早入门三年
- **表面**：天真活泼，喜欢黏着玩家
- **真实**：云深子安插在弟子中的眼线，监视所有人
- **动机**：被云深子收养长大，对其绝对忠诚
- **与玩家关系**：复杂的感情——任务与真心的冲突

### 3.7 护宗灵兽：雪团

- **身份**：宗门护山神兽，一只白狐
- **特殊**：能看穿伪装，是玩家的早期信息来源
- **与玩家关系**：从好奇到信任，关键时刻会帮助玩家

## 4. 章节大纲

### 第一章：入宗

玩家转生醒来，被云深子收为弟子。熟悉宗门环境，结识各位师兄师姐。发现一些"不对劲"的端倪：丹堂的异味、大师兄深夜外出、禁地的低语。

**关键选择**：
- 发现丹堂密室：报告师父 / 暗中调查 / 与红绡交易
- 撞见大师兄修炼禁术：威胁举报 / 装作不知 / 表示理解

**Boss 战**：大师兄的试炼——一道剑意分身，测试玩家实力
