# 剧情与角色设计

## 1. 故事梗概

玩家扮演一名现代大学生林疏影，在一次雷雨夜被闪电击中，意识模糊间转生到修仙世界，成为天剑宗刚入门的小师妹。

天剑宗位于云海深处的浮空山脉，表面上是正道魁首，镇压魔道、庇护苍生。然而随着玩家逐渐深入宗门生活，发现全宗上下皆是"反派"：
- 大师兄萧寒修炼禁术《噬灵剑诀》，以生灵精血铸剑
- 二师姐红绡掌管丹堂，以修士炼制人丹
- 师父云深子竟是千年前被封印的魔道巨擘"云深魔君"

系统任务是"成为正道之光"，但身边全是反派。玩家需要在"维持正派 façade"与"揭露真相"之间做出选择，最终走出自己的"剑道"。

---

## 2. 核心主题

- **剑无正邪，唯心所向**：呼应标题"不知剑"，强调选择的意义
- **伪善与真恶**：正道与魔道的界限是否如此分明？
- **系统与自由意志**：被强加的"正道任务"vs 自己的选择
- **身份与认同**：转生者的归属困境——前世记忆与今世羁绊

## 4. 主要角色

### 4.1 玩家角色：林疏影

- **身份**：转生者，天剑宗小师妹
- **性格**：由玩家选择塑造（初始为好奇、谨慎）
- **特殊能力**：
  - **因果可视**：作为转生者，能看到他人头顶的"因果线"（颜色代表善恶：白=善，红=恶，紫=灰色）
  - **前世记忆碎片**：特定道具或场景可触发前世片段
- **秘密**：前世是一名修士，与云深魔君有莫大渊源——曾是其道侣，在千年前的大战中陨落
- **初始属性**：体质 8/20，灵根 水(6)火(4)，剑心 50（中立），道德 50（中立）

### 4.2 师父：云深子 / 云深魔君

- **表面**：仙风道骨的正道宗师，深受弟子爱戴
- **真实**：千年前被正道联手封印的魔道巨擘"云深魔君"，借尸还魂建立天剑宗
- **动机**：建立天剑宗是为了积蓄力量，向当年封印他的正道复仇（正道诸圣）
- **对玩家的态度**：似乎知道玩家的转生身份，有意培养——是觊觎玩家的特殊体质，还是另有所图？
- **关键剧情**：结局篇会揭示他最终的目标——打破人仙两界壁垒，夺取仙界资源
- **好感度影响**：
  - 高好感：传授真正剑道，最终给予玩家"魔君传承"
  - 低好感：将玩家视为棋子，最终在结局中与玩家对峙

### 4.3 大师兄：萧寒

- **表面**：冷峻正直，宗门楷模
- **真实**：修炼禁术《噬灵剑诀》，每杀一人便强一分，剑身以生灵精血淬炼
- **动机**：幼时全族被魔道所灭，发誓要以任何手段变强报仇
- **身世**：萧家村灭门案背后另有隐情——凶手可能与正道有关
- **与玩家关系**：
  - 初期冷漠，若玩家选择理解他则逐渐敞开心扉
  - 关键：可选择帮助他复仇（获得强大剑意技能）或阻止他（触发分支）
- **个人线**：通过多次对话和特定道具（萧家遗物），可解锁其完整背景

### 4.4 二师姐：红绡

- **表面**：温柔美艳，擅长炼丹救人
- **真实**：以修士炼制人丹，甚至将失败品"赐"给外门弟子
- **动机**：天生绝脉无法修炼，只能靠人丹续命延寿
- **身世**：曾是某个小宗门的掌上明珠，全家被天剑宗"招收"时惨遭炼成丹药
- **与玩家关系**：
  - 表面热情，实则试探玩家是否"可用"
  - 若玩家威胁到她则毫不留情（触发敌对战斗）
  - 高好感：愿意告诉玩家她真正的身世，并交出关键线索道具

### 4.5 三师兄：墨言

- **身份**：阵法天才，掌管宗门禁地
- **性格**：书卷气重，不善言辞，似乎知道很多但不说
- **立场**：中立派，既不认同也不揭发，只守护自己在乎的人
- **与玩家关系**：最容易成为盟友的角色，但需要证明玩家的"不同"
- **关键剧情**：可带玩家进入后山禁地核心区域（需通过阵法考验）
- **个人线**：其身世与上古剑冢有关，是"剑冢守护者"血脉的后裔

### 4.6 四师姐：白芷

- **身份**：最小的师姐，比玩家早入门三年
- **表面**：天真活泼，喜欢黏着玩家
- **真实**：云深子安插在弟子中的眼线，监视所有人
- **动机**：被云深子收养长大，对其绝对忠诚，认为"师父做的一切都是对的"
- **与玩家关系**：复杂的感情——任务与真心的冲突
- **个人线**：随着剧情推进，她会逐渐发现云深子的某些秘密，内心产生动摇

### 4.7 护宗灵兽：雪团

- **身份**：宗门护山神兽，一只白狐（九尾血脉）
- **特殊能力**：能看穿伪装（破除幻术和易容），是玩家的早期信息来源
- **与玩家关系**：从好奇到信任，关键时刻会帮助玩家
- **背景**：千年前曾被云深魔君所救，与魔君有契约在身，是魔君设置的后手之一

## 5. 章节大纲

### 第一章：入宗

**时长**：约 2-3 小时游戏时间
**主线目标**：熟悉天剑宗，发现"不对劲"的端倪
**触发条件**：游戏开场，自动进入

#### 主要场景
1. **天剑宗山门** — 醒来，被云深子收为弟子
2. **主殿（天剑殿）** — 拜师仪式，领取弟子令牌
3. **弟子居所** — 安顿，结识师兄师姐
4. **后山（入口）** — 第一次探索触发

#### 关键事件

| 顺序 | 事件 | 触发条件 | 结果 |
|------|------|----------|------|
| 1 | 转生苏醒 | 开场 | 云深子收为弟子，获得令牌 |
| 2 | 认识大师兄 | 进入大殿 | 萧寒冷淡接待，态度冷漠 |
| 3 | 认识二师姐 | 进入大殿 | 红绡热情，但玩家可见其"因果线"为红色 |
| 4 | 认识三师兄 | 弟子居所 | 墨言点拨玩家："多看少说" |
| 5 | 认识四师姐 | 弟子居所 | 白芷缠着玩家，内心os可见"眼线"提示 |
| 6 | 雪团初遇 | 后山入口 | 白狐开口说话，暗示知道玩家身份 |
| 7 | 外门弟子失踪 | 日常任务 | 周铁柱未出现，引发调查 |
| 8 | 丹堂异味 | 靠近丹堂 | 闻到血腥味，可选择调查 |
| 9 | 大师兄禁术 | 夜间跟踪/特定时间 | 撞见萧寒修炼噬灵剑诀 |
| 10 | Boss战 | 与萧寒对话后 | 萧寒剑意分身试炼 |

#### 关键选择

| 选择点 | 选项 | 后续影响 |
|--------|------|----------|
| 发现丹堂密室 | A. 报告师父 | 道德+5，云深子表示会处理，但密室消失 |
| | B. 暗中调查 | 解锁红绡支线，获得线索道具 |
| | C. 与红绡交易 | 获得丹药，好感度红绡+10，道德-5 |
| 撞见大师兄修炼 | A. 威胁举报 | 萧寒敌对，后续剧情锁死，道德+5 |
| | B. 装作不知 | 安全，但错过深入了解萧寒的机会 |
| | C. 表示理解 | 萧寒好感+10，解锁萧寒个人线前置 |
| 外门弟子失踪 | A. 调查到底 | 解锁支线任务「无声之死」 |
| | B. 听从官方说法 | 获得少量经验，但错过真相 |
| | C. 向白芷打听 | 白芷好感+5，但白芷会上报云深子 |

#### Boss 战：大师兄的试炼

- **类型**：教学战，必败/可胜
- **敌人**：萧寒剑意分身（半透明剑影形态）
- **机制**：只允许使用基础剑招，积累剑意槽
- **可触发条件**：与萧寒对话选择"想变强"后触发
- **失败结果**：被击晕，获得少量经验，了解萧寒真正实力
- **胜利奖励**（高难度）：萧寒刮目相看，解锁《噬灵剑诀》入门式

#### 第一章结尾

玩家回到弟子居所，发现桌上多了一封信（来源不明），信中暗示：
- 云深子不是他表面那样
- 天剑宗的历史远比想象的复杂
- 附一张残缺的地图碎片

### 第四章：抉择

**时长**：约 2-3 小时游戏时间
**主线目标**：最终结局分支，走向自己的剑道
**触发条件**：完成第三章所有主线事件

#### 最终决战路线分支

根据玩家的**剑心值**、**道德值**、**关键道具**、**好感度**综合判定

| 结局 | 触发条件 | 最终Boss | 核心选择 |
|------|----------|----------|----------|
| 剑心通明 | 剑心≥80，揭露真相，拯救宗门 | 云深子（本体） | 以己为剑，斩断因果 |
| 云深归处 | 道德≥80，举报全宗 | 无（和平结局） | 正道制裁，天剑宗重建 |
| 不知处 | 道德≤-80，认同反派 | 正道联军 | 魔君传承，与天下为敌 |
| 轮回 | 持有全部轮回碎片 | 内心之敌（自己） | 发现轮回真相，选择重新开始 |
| 断剑 | 所有好感度≤20 | 无（孤独结局） | 孤身离开 |

#### 结局详情

**剑心通明（真结局）**
- 玩家将自身剑意推到极致，以"剑心通明"状态与云深子对决
- 揭示玩家前世正是破解魔君封印的关键——当年封印魔君的正是玩家前世
- 魔君千年的布局最终是为了等玩家转生归来，完成未竟的契约
- 玩家可以选择：斩杀魔君并重建宗门 / 与魔君和解，化解千年恩怨
- 解锁"后日谈"：宗门在新秩序下繁荣，玩家成为新的宗主

**云深归处（正道结局）**
- 玩家向正道诸宗揭发天剑宗真相
- 经历大战后天剑宗被正道联军制裁，魔君被再次封印
- 玩家成为正道新星，但内心知道有些师兄师姐并非坏人
- 解锁"遗憾"后日谈：红绡的孩子（若救下）长大后的故事

**不知处（魔道结局）**
- 玩家继承云深魔君衣钵，以"新魔君"身份君临天下
- 与正道开战，血流成河
- 解锁"代价"后日谈：玩家逐渐失去人性，变成真正的"魔"

**轮回（隐藏结局）**
- 玩家发现整个故事是一个轮回——玩家曾经历过无数次类似的循环
- 每次轮回都以不同方式失败或成功
- 最终选择：打破轮回，回到最初转生的那一刻重新开始
- 解锁"无限"后日谈

**断剑（隐藏结局）**
- 所有角色好感度极低，无人信任玩家
- 玩家孤身离开天剑宗，浪迹天涯
- 开放式结局，暗示玩家在流浪中逐渐找回自己
