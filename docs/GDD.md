# 《云深不知剑》Game Design Document

## 1. 游戏概述

| 项目 | 内容 |
|------|------|
| **游戏名称** | 云深不知剑 |
| **英文代号** | SwordInTheClouds |
| **副标题** | 我转生成了剑宗小师妹，但全门派都是反派？ |
| **类型** | 2D 像素风剧情向回合制 RPG |
| **平台** | 浏览器（PC / 移动端自适应） |
| **技术栈** | Phaser 3 (v3.70+) + TypeScript 5.0 + Vite |
| **基础分辨率** | 320×180（像素完美渲染，自适应整数倍缩放） |
| **游戏时长** | 主线 8–12 小时 |
| **结局数量** | 3 个主要结局 + 2 个隐藏结局 |

## 2. 世界观与设定

玩家扮演一名现代大学生，意外转生到修仙世界，成为**天剑宗**刚入门的小师妹。

宗门位于云海深处的浮空山脉，表面上是正道魁首，实则全宗上下皆为"反派"：
- **大师兄**：修炼禁术《噬灵剑诀》
- **二师姐**：以修士炼人丹，掌管丹堂
- **师父**：千年前被封印的魔道巨擘"云深魔君"

玩家需要在"维持正派 façade"与"揭露真相"之间做出选择，最终影响整个修仙界的命运。

### 核心叙事张力
- 系统任务是"成为正道之光"，但身边全是反派
- 道德抉择：是否举报同门？是否利用反派资源变强？
- **剑道真意**：剑无正邪，唯心所向（呼应标题"不知剑"）

## 3. 核心玩法机制

### 3.1 探索系统（Overworld）
- 2D 俯视角像素地图，支持八方向移动
- 可交互对象：NPC、宝箱、传送点、修炼静室
- 迷雾机制：部分区域需要特定剧情或道具解锁
- 昼夜循环：影响 NPC 位置和特殊事件触发

### 3.2 回合制战斗系统（ATB 变体）
- **触发方式**：明雷遇敌（可见敌人巡逻）+ 剧情强制战斗
- **战斗队伍**：主角 + 最多 2 名同门（随剧情解锁）
- **行动顺序**：基于"身法"属性的速度条系统（Active Time Battle）
- **行动类型**：普通攻击、剑诀（技能）、道具、防御、逃跑
- **五行相克**：金木水火土属性克制关系（伤害 ±30%）
- **连携系统**：特定角色组合触发合击技

### 3.3 角色成长系统
- **基础属性**：气血、灵力、攻击、防御、身法、悟性、根骨、机缘
- **境界系统**：炼气 → 筑基 → 金丹 → 元婴 → 化神（每境分前/中/后/圆满）
- **功法修炼**：装备不同功法改变战斗定位和属性成长倾向
- **剑心系统**：独立经验条，通过道德抉择和剧情事件提升，影响可用技能池
- **好感度系统**：与 6 位主要角色的好感度影响剧情分支和合击技解锁

### 3.4 剧情对话树系统
- **分支结构**：每个对话节点包含 2–4 个选项
- **属性检定**：部分选项需要悟性/机缘/好感度达标
- **隐藏选项**：满足特定条件（携带某物品、完成某支线）后解锁
- **时间限制**：关键抉择添加倒计时，增加紧张感
- **因果簿**：所有重大选择记录到"因果簿"，可在菜单查看

### 3.5 物品栏系统
- **分类**：武器、防具、饰品、丹药、材料、秘籍、任务道具
- **容量**：背包 48 格 + 专属材料袋（999 堆叠）
- **装备系统**：武器 + 衣服 + 鞋子 + 饰品 × 2
- **合成系统**：丹炉/锻造台合成高级物品（需要配方）
- **物品来源**：战斗掉落、地图探索、NPC 交易、任务奖励

## 4. 多结局设计

| 结局名称 | 路线 | 达成条件 |
|----------|------|----------|
| **剑心通明** | 真结局 | 剑心 100，揭露真相但拯救宗门 |
| **云深归处** | 正道线 | 举报全宗，成为正道领袖 |
| **不知处** | 魔道线 | 认同反派，继承魔道衣钵 |
| **轮回** | 隐藏 | 特定道具组合触发前世记忆 |
| **断剑** | 隐藏 | 好感度全灭，孤身离开 |
