> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

## Task 6: OverworldScene — 实现传送门状态判定和视觉反馈

**Files:**
- Modify: `src/scenes/overworld-scene.ts`

需要在创建传送门对象时解析其状态，并应用视觉反馈。

- [ ] **Step 1: 添加状态判定方法**

在 `OverworldScene` 类中添加私有方法：

```typescript
  private resolveTeleportStatus(obj: MapObject): { status: 'available' | 'locked' | 'conditional'; hint?: string } {
    // 1. Explicitly locked
    if ((obj as any).locked === true) {
      return { status: 'locked', hint: '禁制未解，不可擅入' };
    }

    // 2. Target map not loaded
    const target = (obj as any).target as string;
    if (target && !this.loadedMapIds.has(target)) {
      return { status: 'locked', hint: '此方天地尚未开启' };
    }

    // 3. Condition not met
    const condition = (obj as any).condition as { type: string; hint: string; itemId?: string; flag?: string } | undefined;
    if (condition && condition.type !== 'none') {
      // TODO: check actual inventory/flag state when those systems are wired
      // For now, always show conditional hint if condition exists
      return { status: 'conditional', hint: condition.hint || '灵气阻隔，难以逾越' };
    }

    // 4. Available
    return { status: 'available' };
  }
```

- [ ] **Step 2: 在创建传送门时解析状态并应用视觉反馈**

找到创建传送门 visual 的代码（在 `loadMap` 或 `createMapObjects` 附近），在创建 portal sprite 后添加：

```typescript
    const zoneState = this.resolveTeleportStatus(obj);
    this.teleportZones.set(obj.id || `${obj.x}-${obj.y}`, zoneState);

    // Apply visual tint based on status
    if (zoneState.status === 'locked') {
      portalSprite.setTint(0x888888);
      portalSprite.setAlpha(0.5);
      // Stop animation if applicable
      if (portalSprite.anims.isPlaying) {
        portalSprite.anims.pause();
      }
    } else if (zoneState.status === 'conditional') {
      portalSprite.setTint(0xffaa44);
      // Slow animation if applicable
      if (portalSprite.anims.isPlaying) {
        portalSprite.anims.timeScale = 0.5;
      }
    }
```

注意：`portalSprite` 需要根据实际代码中的变量名调整。如果传送门 visuals 是由 `map-loader.ts` 创建的，可能需要在 `OverworldScene` 中找到对应的 sprite 引用方式。

## Task 7: OverworldScene — 实现场景切换守卫和 showFloatingHint

```typescript
  private transitionToScene(sceneKey: string, x: number, y: number, targetMapId?: string): void {
    const mapId = targetMapId ?? this.currentMapId;
    if (sceneKey === 'OverworldScene' && !this.loadedMapIds.has(mapId)) {
      this.showFloatingHint('前路隐于迷雾之中，似有大能设下的禁制');
      return;
    }
    this.cameras.main.fadeOut(300, 0, 0, 0);
    this.cameras.main.once(Phaser.Cameras.Scene2D.Events.FADE_OUT_COMPLETE, () => {
      this.scene.start(sceneKey, { mapId, playerX: x, playerY: y } as SceneTransitionData);
    });
  }
```

在 `OverworldScene` 类中添加：

  private showFloatingHint(text: string): void {
    if (!this.player) return;
    const hint = this.add.text(this.player.x, this.player.y - 24, text, {
      fontSize: '10px',
      color: '#ffaa44',
      stroke: '#000000',
      strokeThickness: 2,
    });
    hint.setOrigin(0.5);
    hint.setDepth(50);

    this.tweens.add({
      targets: hint,
      y: hint.y - 16,
      alpha: 0,
      duration: 2000,
      ease: 'Power1',
      onComplete: () => hint.destroy(),
    });
  }
